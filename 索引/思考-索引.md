# 组件实现原理
#### 一个有状态的组件就是一个选项对象
```js
const MyComponent = {
  name: 'MyComponent',
  data() {
    return {
      foo: 1
    }
  }
}
```
#### 用虚拟节点描述组件
```js
// type 属性存储组件的选项对象
const vnode = {
  type: MyComponent,
  // ...
}
```
#### 一个组件必须包含一个渲染函数，即 render 函数，并且返回值是虚拟 dom。
```js
const MyComponent = {
  name: "MyComponent",
  render() {
    return {
      type: "div",
      children: "text",
    };
  },
};
```
